# Промпт для Claude Code

Создай веб-сервис для быстрой загрузки изображений в S3-хранилище Timeweb Cloud с автоматическим копированием ссылки.

## Стек

- Frontend: HTML + CSS + vanilla JS (один файл index.html)
- Backend: Node.js + Express
- S3: библиотека @aws-sdk/client-s3 для работы с Timeweb S3-хранилищем

## Подключение к S3

- Endpoint: https://s3.twcstorage.ru
- Region: ru-1
- Bucket: my-data (публичный)
- Credentials хранятся в .env файле (S3_ACCESS_KEY, S3_SECRET_KEY)
- Формат публичной ссылки после загрузки: https://my-data.s3.twcstorage.ru/{имя_файла}

## Функциональность

1. Drag-and-drop зона для загрузки изображений (+ клик для выбора файла)
2. При загрузке — показывать прогресс-бар
3. Имя файла: транслитерировать кириллицу в латиницу, заменять пробелы на дефисы, добавлять короткий уникальный суффикс (6 символов) чтобы файлы не перезаписывались. Пример: moy-banner-a3f8d1.png
4. После успешной загрузки:
   - Показать превью загруженного изображения
   - Показать готовую ссылку
   - Автоматически скопировать ссылку в буфер обмена
   - Кнопка "Скопировать ссылку" для повторного копирования
5. История загрузок в текущей сессии — список всех загруженных файлов с превью, ссылками и кнопками копирования
6. Поддержка форматов: jpg, jpeg, png, gif, webp, svg
7. Максимальный размер файла: 10 МБ
8. Возможность загрузить несколько файлов одновременно

## Дизайн

- Минималистичный, тёмная тема
- Адаптивный (работает и на телефоне)
- Drag-and-drop зона с пунктирной рамкой, при наведении файла — подсветка
- Уведомления об успехе/ошибке

## Важно

- Сервер должен запускаться одной командой: npm start
- При первом запуске — проверять наличие .env и выдавать понятную ошибку, если ключи не заполнены
- Добавить CORS-заголовки
- Не загружать файлы через S3 presigned URL, а проксировать через свой сервер (чтобы не светить ключи на фронтенде)
